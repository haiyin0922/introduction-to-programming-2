# Mini Project 1

1. "Mini project 1.pptx" 是 mini-project-1 的介紹。

2. "calculator_loop" 和 "calculator_recursion" 是我們提供的 packages. 詳情請參考 "Mini-Project 1 - package 解說.pptx".

3. 完成 project 後，可以使用 "miniproject parser" 測試。測資放在 "parser test case". (parser 會在之後公佈)

4. 請同學注意："parser test case" 裡的測資中，x, y, z 的初始值皆為 0, 但 demo 時並不會初始為 0, 同學必須從 memory 中把它們的初始值存取出來。可以參考 "Mini project 1.pptx" 投影片的第 7 頁。

5. 記憶體不會發生超過 256 bytes 的錯誤。
==============================================
1. "Mini project 1.pptx" is the introduction to mini-project-1.

2. "calculator_loop" and "calculator_recursion" are the provided packages. Please refer to "Mini-Project 1 - package 解說.pptx" for details.

3. After you finish the project, you can test it with "miniproject parser". The test cases are in "parser test case". (parser will be announced in the future)

4. Note that x, y, z's initial values are all 0 in the test cases of "parser test case". However, they won't be initialized as 0 in the demo. You have to take the initial values from memory. You can refer to page 7 of "Mini project 1.pptx"

5. There will be no cases of memory exceeding 256 bytes.
